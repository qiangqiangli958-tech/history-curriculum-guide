# 05学业水平考试命题要求

05学业水平考试命题要求

## 一、学业评价与学业水平考试
在义务教育阶段的历史课程学习中，学业评价始终相伴，如影随形。而一般意义上的、通俗理解的“考试”,则是学业评价的手段之一，或者说是最常见、最重要的手段。但是在《义务教育历史课程标准(2022年版)》中，“学业水平考试”则有严格的界定：历史课程学业水平考试，是依据内容要求、学业要求和学业质量标准，对学生完成历史课程后的课程目标达成度进行终结性评价的考试。考试结果是衡量学生是否达到毕业标准和高一级学校招生录取要求的重要依据。也就是说，课标中的学业水平考试乃是特指，性质是“终结性评价”,它是衡量考生是否达到毕业要求的标尺，是考生能否进入高一级学校继续学习的重要依据。

关于学业评价与学业水平考试的关系，可以从以下两方面理解。

(1)两者存在一定关联。

其一，它们都是历史课程实施的重要环节。学业评价是对学生日常的、不间断的课程素养达成度的测量，学业水平考试则是最终的、综合性测量。某种程度上说，学业水平考试是系列学业评价的总括和终结。

其二，它们的指导思想和原则完全一致，即遵照课标总体要求，坚持以德为先、 能力为重、全面发展，“体现历史课程的育人功能，培养学生的核心素养，引导学生初步树立正确的历史观、民族观、国家观、文化观.明理、增信、崇德、力行”。另 外，课标设定的学业质量(学业质量标准)也都是学业评价和学业水平考试(历史课程 素养水平达成度)的唯一测量标准。两者区别仅在于，课程评价对学业质量的运用， 可以是分时段、分层次、多维度、循序渐进的，而学业水平考试则是整体贯彻，一次性统一考查。

其三，它们对于“教”与“学”都有一定引导和指向作用，都可以作为评价教学质量的参考，也可以为教师改进教学提供借鉴。相比较而言，学业水平考试在中学的关注度更高一些。

(2)两者存在一定差异。

其一.两者的地位不同。学业评价是义务教育中教学链条的中间环节，具有变 动与发展的特点，多数情况下教学指导的参考性大于权威性；学业水平考试则是义务教育阶段教学的终结，测量结果具有权威性和标志性，一般情况下不容更改。

其二，两者的实施目标不同。课程评价主要是测量学生日常积累与表现出的核心素养水平，评价结果主要用于改进教师教学和学生学习，促使学生核心素养不断 增进和发展；而学业水平考试则是指标性的，考试成绩是衡量学生是否达到毕业标准、是否达到高一级学校招生录取要求的重要依据.重要性远远超过一般学业评价。

其三，两者的实施手段存在差异。学业评价和学业水平考试虽然都强调测量的科学性、专业性、客观性和有效性，但学业评价紧紧依托教学实际和学生实际，能够运用的手段和方式不仅丰富，而且灵活；学业水平考试实施手段相对单一。就目前我国科技教育与测量手段和义务教育实际情况而言，一次性、定时纸笔测试仍是学业水平考试的最主要方式。

其四，两者对教学的引导作用不同。学业评价是对某一特定学生群体义务教育 阶段历史课程学习的长时段跟踪与测量，可以不间断地对教师的教与学生的学进行 针对性引导或指导；学业水平考试对某一特定学生群体而言则是终结性的，它对教 学的引导或指导作用主要体现在教师的经验积累与总结。这些经验主要是对下一年 级或批次的师生起引导作用。

## 二、学业水平考试与考试命题
学业水平考试是对学生义务教育阶段历史课程学习后目标达成度的终结性考查.通过试题与试卷形式实施、完成。关于考试与命题，需注意以下几个问题。

(1)学业水平考试的性质、原则与标准等，由《义务教育历史课程标准(2022年版)》明确规定，具有权威性，试题命制必然要以课标为基准。这就要求命题人对课 程标准有整体、深刻的领悟，准确把握其主旨与神髓。只有这样，命制出的试题才 能不背离国家的教育方针，准确把握与体现育人与选人标准。

(2)与原有课标相比，新修订的《义务教育历史课程标准(2022年版)》提出并明确强调历史课程要培养的核心素养，明确制定了学业质量，这就要求试题命制人员 对核心素养及学业质量有准确的理解与把握，从试题内容、形式到测评标准都要进行探索与创新，使学业水平考试整体符合与适合考查核心素养的要求。

(3)《义务教育历史课程标准(2022年版)》是国家文件，学业水平考试是政府行为，其中相关规定多是原则性和指导性的。学业水平考试试卷和试题在不违背上述原则的前提下，还应体现自己的特性与特点。针对不同年份、不同地域、不同时代背景、不同学生实际学习情况，试题应体现出时代性、地域性与文化性差异。这种试题差异对义务教育阶段历史课程要培养的核心素养的测量具有积极作用。另外，不同特色的试题间相互比较与借鉴也是教学相长、相互促进提高的有效方式。

## 三、试题命制的指导原则
学业水平考试与命题指导原则必须与《义务教育历史课程标准(2022年版)》的规  定相一致，即突出历史课程性质和特点，考查课程目标实现程度和核心素养达成度。 这些指导原则主要包括以下几点。

(一)全面贯彻党和国家的教育方针，切实落实立德树人根本任务

立德树人是历史课程最基本、最重要的教育理念，也是历史课程学业水平考试 所遵循的第一原则。命制试题绝不能违背国家大政方针，要确保政治正确。这是因为历史课程本身具有极强的意识形态导向。“历史学是在一定的历史观指导下叙述和阐释人类历史进程的学科。马克思主义指导下的历史学，以探寻历史真相、总结历 史经验、认识历史规律、认清历史发展趋势为其重要功能。”“义务教育历史课程是学 生在马克思主义唯物史观指导下，了解中外历史发展进程、传承人类文明、提高人 文素养的课程，具有思想性、人文性、综合性、基础性特点，具有鉴古知今、认识历史规律、培养家国情怀、拓宽国际视野的重要作用。”

(二)注重历史课程独有的育人功能

与其他义务教育课程相比，历史课程自有其育人特点与优势。例如，它的内容涵盖古今中外，包括政治、经济、思想、文化、社会多个领域；集知识性、思想性、 教育性与趣味性于一身；科学揭示了人类社会发展的客观规律，展现出国际的、多元的民族地域文化；与其他课程知识与素养存在密切交叉与交融；等等。另外，历史课程还具有科学性与人文性相统一的特点，历史资料或历史材料不仅种类繁多， 可读性强，而且真实、可信。所有这些对培养学生正确的人生观、世界观、价值观， 特别是历史观、民族观、国家观、文化观，增强社会担当，德智体美劳全面发展等，都能起到其他课程无可取代的作用。学业水平考试命题也应重点考查这些内容与内涵，体现历史试题的特点与特色。

(三)注重考查历史课程要培养的核心素养的达成程度

历史课程要培养的核心素养的提出是新的课标的重大变化和重点，自然也是学业水平考试的重要落脚点。需要指出的是，学业水平考试不能仅仅停留在对唯物史观、时空观念、史料实证、历史解释、家国情怀五个方面的表面考查，还应深入其中，即不是简单的“涉及”,而是有明确指向，即试题明确考查各素养的具体内涵， 也就是学业质量。

## 四、试题命制的思路、步骤和框架
学业水平考试试题命制是“系统工程”,需要较长时段的思考、酝酿和准备。从明确任务到试卷完成，需要注意以下几方面。

(一)命题思路

学业水平考试命题应从发展学生核心素养的角度制定考试目标，将核心素养的培养作为命题出发点和落脚点。命题人员应不断学习《义务教育历史课程标准 (2022年版)》和现有、现用的历史课程教材，制定的考试目标和选择的考试内容要 符合国家政策和课标要求，避免出现超出课标要求等问题。

制定考试目标和考试内容还应顾及当时、当地的实际情况，使试题和试卷具有 更强的针对性和指向性，避免新命制的试题与以往试题大规模的雷同或重复，保证 核心素养的考查真实、有效。

(二)命题步骤和框架

试题命制思路确定后即开始具体的试题命制。从命题到完成组卷大致包括如下 步骤：

依据考试目标制定试卷初步蓝图，并随命题进展进行微调。

搜集试题素材，梳理打磨成为试题毛坯。

确定试卷结构与试题形式，将试题毛坯修改成题。

依据实际情况不断微调试卷与试题结构与形式，最终完成试题和试卷。

制定评分标准、参考答案、答案示例或答案参考。

依据最终试题、试卷和评分标准，预估试题与试卷相关参数，准备与考试实际 情况相对照，反思能否实现最初设定的考试目标。

(三)命题步骤的关键

1.试卷蓝图

考试目标确定后，一般需制定此次考试的试卷蓝图。试卷蓝图是考试目标的具体细化与落实，也是组题成卷的总纲。蓝图是否合理、可行，是试题命制成功与否的关键。

试卷蓝图一般包括以下内容：考试目标、试题类型与数量、考试时间、细化的  考试内容、试题样例、历史课程核心素养在试题中的分布与体现、相关参数预估等。上述内容需在蓝图中综合考量，整体布局。其中最主要的就是考试内容的细化和历史课程核心素养的分布。

考试内容在《义务教育历史课程标准(2022年版)》与现有教材有明确规定，包括：中国古代史、中国近代史、中国现代史，世界古代史、世界近代史、世界现代史以及跨学科主题学习共七个板块，涉及不同历史时期的事件、人物、制度、生产、生活、思想文化等方面的史实。因此，试题内容一般应尽量选择、涵盖上述史事、制度和观念，注重考查“人类社会从分散到整体、从低级到高级的发展历程，初步把  握中外历史发展的基本线索，认识不同历史时期的时代特征”。另外，对历史发展线索、历史发展规律和不同文化、文明间的简单比较，也是学业水平考试的重要考查内容。

需要特别强调的是，学业水平考试应注重于强化对核心素养(唯物史观、时空观念、史料实证、历史解释、家国情怀)水平达成度进行考查，这是以往课标和考试中 没有的新因素，在制定蓝图时应主动将其作为考查规划的重点。比如哪些内容或试 题适合考查哪一个(或几个)核心素养；哪个试题形式适合考查哪个核心素养的哪些 具体内容(学业质量);核心素养水平考查的广度、深度与梯度如何布局；等等。

对上述几方面内容的总策划就是制定考试蓝图的重点，其关键在于确定考查重点内容与平衡各方面的比例关系。比如，七个模块的考查内容和赋分比例；政治、 经济、思想文化和社会生活等方面的考查内容和赋分比例；五个方面核心素养水平考查的侧重点与分布；等等。总体策划既要综合平衡这几方面的比例关系，使试卷协调统一，试题内容既不重复又不抵牾，同时又重点突出，特色鲜明，富有时代感。

2.试题素材的搜集、剪裁与整理

搜集试题素材是长期积累的过程。学业水平考试蓝图制定后，试题素材的剪裁与 整理就成为蓝图能否实现的关键。再美好的蓝图也需要通过一道道的试题得以实现。

第一，“新情境”的营造是素材选择与试题命制的立足点与突破口。

创设问题情境是考试测量常用的方式，也是寻找试题素材的目的。 一般来说，考查一个学段学业水平的终结性考试，或者具有选拔功效的学业水平考试，其试题  尽量不宜采用以往的“陈题”,也尽量不使用日常历史教学和学业测量中经常使用的素材。之所以如此“苛刻”,是因为学业水平考试必须贯彻与体现《义务教育历史课程  标准(2022年版)》的要求，即“题目命制时要创设新情境，可以从学习、生活、社会  等方面选取素材”。这是以往义务教育考试中不曾有过的硬性规定。为何创造?如何  创造?命题人必须对此有足够的认识。

创设新情境与历史课程核心素养考查紧密相关。学业水平考试命题应从发展学生核心素养的角度制订考试目标，将核心素养培养作为考试命题的出发点和落脚点。 核心素养是学生通过课程学习逐步形成的正确价值观、必备品格和关键能力。也就  是说考查核心素养主要不是考查学生已经“固有”的知识、结论或答题模式，而是考  查学生通过历史课程学习后自身已具备的观念、品格和能力水平.即学业质量。具  备一定核心素养的学生应该能够依托已有知识，独立思考并初步解决以往没有遇到  过的问题，初步应对新的挑战。学业水平考试试题采用对学生来说相对“陌生”的新  情境构建和命制试题，可以最大限度地避开学生和教师的应试备考，较为真实地测 量出学生自身具备的核心素养水平。

第二，新情境的选择与营造应适宜、适度。在义务教育阶段，学生历史知识掌 握不多，核心素养刚刚培育，学科间综合能力和分析问题、解决问题的关键能力不 强，因此设置学业水平考试新情境要适合学生实际。课标指出，学业水平考试新情 境“所选素材要与学生所学内容、具备的关键能力具有一定关联。设置的问题、任务 要与新情境高度融合，指向明确，符合学生的认知水平”。如果情境太新或太偏，学 生无法与所学相联系，难以做到知识的“迁移”或者超出学生核心素养水平，也无法 实现考试目标。

第三，试题素材剪裁与整理要符合学生学习实际。

运用新素材营造新情境除了要适宜、适度外，还要符合学生学习实际情况，即 考虑学生整体的、综合的素养水平和认知能力，特别是义务教育阶段其他课程的核 心素养水平，不能因为使用新素材而给学生制造太多障碍。例如，使用原始史料时 要考虑学生的文言文水平，生僻的字、词要有注音和简要说明，较长且难懂的古文 史料可以考虑转化为白话文资料，尽量使用地图、图表等手段简化资料烦琐表述， 等等。这样新素材才是新情境而非“新障碍”。

3. 试卷结构与试题形式

试卷结构主要指试卷的试题组成与方式。影响试卷结构的因素包括考试目标、 蓝图规划和考试时间等。试题形式大致分为两类，即客观题和主观题(从作答标准  看),或称选择题和问答题(从作答形式看)。

客观题或选择题的试题素材相对简练，设问指向明确，分值较小但覆盖面广，适合考查学生的识记、认知与理解能力；其局限性在于试题给予学生自主发挥的余地不大，对核心素养水平的考查辨识度有限。

主观题或问答题素材使用量大，信息包容丰富，设问形式多种多样，设问指向 各有侧重，如“简答”“概括”“概述”“论述”“说明”“分析”“比较”“解读”“评析”等。该 题型主观性与开放性的特点能给予学生较多自主发挥余地，对核心素养水平的考查较为有益。

主观试题一般分值较大， 一道试题也可分类、分层设问，遵循由易到难原则层 层递进，逐渐深入。例如试题设计可为三个层次： 一是初级层次，设问相对直接、简单，主要考查学生对已有历史认知成果的记忆和理解；二是中级层次，设问相对 复杂，重点考查学生运用历史学科思想方法和所学知识概括、分析历史问题的能力； 三是高级层次，设问相对开放，多联系社会和学生实际设置情境和问题，考查学生 综合运用各种史学方法解决相对复杂的历史问题能力。在义务教育阶段，对第3层 次的考查比例不宜过多。

总之，确定试卷结构最主要的内容就是依据考试目标、考试蓝图、学生实际和 考试时间，确定客观题与主观题的比例及分数；确定客观题或主观题的各自数量及 分数分配；确定客观题和主观题各自的难度系数；等等。再次强调的是，客观题和 主观题尤其是主观题，一般都需设置新情境。

4. 参考答案与评分标准

客观题或选择题由于设问指向明确，答案相对客观、唯一，不存在疑义，答案 分为单选或多选， 一般情况下通过设置A 、B 、C 、D 选项即可实现。

较高层次的主观题设问因要突出主观性和开放性，故答案设置提倡客观标准而  非“答案唯一”,形式除“标准答案”外，还有“答案要点”“答案示例”“答案参考”等。 “标准答案”一般针对考查内容或知识点比较固定、客观、考生自主选择和发挥余地  不大的设问；“答案要点”“答案示例”或“答案参考”一般是对应对考查内容相对灵活、学生自主选择、发挥空间较大的设问。这类设问难以给出标准答案，只能提供评判、 赋分的标尺与参考，或者是给出赋分标准。

(四)命题过程中需注意的问题

1.历史知识与核心素养要有效链接

知识与素养的关系是历史课程改革经常被提及的重要议题，也是测评与考试不可回避的问题。在义务教育阶段，学生对历史基础知识的掌握十分重要，离开历史知识核心素养无法养成，素养达成度也无法测量；历史课程讲授与考查若仅仅停留 在知识记忆与掌握层面，就背离了新课标的基本理念和要求。历史试题的命制神髓 是将两者紧密结合、有机融合，既要考查历史基础知识又要测量核心素养，以新情 境、新材料迁移学生的既有知识，以素养考查为最高和最终皈依。这是新课标背景下历史考试命题的核心。这是学业水平测试的重点，也是一个难点，更是一个必须突破的探索过程。

2. 局部最佳服从整体最优

好的经典试题既要有新颖的素材情境又能与课标、教材紧密结合，既有巧妙贴切的设问又有理想的难度与区分度。正因为如此，好的试题素材都会令人爱不释手。但在实际操作过程中，命题人一定要分清主次，不能“因材害义”,再好的试题素材  也要服从整个试卷蓝图设计要求，特别是要注重政治性、科学性、试题立意指向、 核心素养分布、教材内容分布、试题适切性、难度系数分布等。这些因素的综合平衡才是衡量试卷水平的最高标准，有时“忍痛割爱”也是一种大局观的正确选择。

3. 注意初中与高中学段核心素养的区别与衔接

历史学科有其自身的、一贯的规律和特点，无论义务教育阶段还是高中教育阶段，历史学科核心素养本质上是一致的，都包含唯物史观、时空观念、史料实证、历史解释、家国情怀五个方面，只是不同学段对核心素养内涵的阐释程度和对学生的素养达成度要求不同而已。课标对核心素养的表述及学业要求相对简洁，学业质 量没有像普通高中历史课程标准那样多层次水平划分。因此在历史试题命制时，对 核心素养内涵的把握要适合初中学生实际，注重考查各素养的核心要义，同时也要注意与高中核心素养的关联，避免出现矛盾与抵牾，便于学段学习转换和素养养成一致性的培育。

4. 关照历史课程素养与其他课程素养的联系

课程核心素养与学生共同素养是个别与一般的关系。一般情况下历史课程核心素养考查不可能完全脱离学生已具备的其他课程核心素养或共通性素养而独立测量。因此在命制历史试题时既要重点考查历史课程核心素养，又要处理好与其他课程素养或共通性素养的关系，既不能因其他课程素养因素影响历史课程核心素养的考查，也不能将历史试题变为考查其他课程素养的试题。
